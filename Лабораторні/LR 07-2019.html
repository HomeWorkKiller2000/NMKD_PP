Проектний практикум. Лабораторна робота № 7

Командна робота над проектом на Github

Мета роботи

Отримати навички роботи в команді репозиторіями на Github

Пояснення до роботи

Дана лабораторна робота проводиться групами в два або більше здобувача освіти

Хід роботи

  1. Створити робочу групу з двох-трьох учасників
  2. Додати через веб-інтерфейс Github соавтора (колаборатора) до свого репозиторію
  3. Прослідкувати, щоб інші учасники вашої робочої групи додали ваш обліковий запис до своїх репозиторіїв
  4. Клонувати репозиторій, до якого вас додали як соавтора
  5. Внести зміни до локальної робочої копії даного репозиторію
  6. Якщо необхідно, додати нові файли до репозиторію
  7. Проїндексувати та зафіксувати внесені зміни
  8. Вивантажити локальну робочу копію на Github
  9. Перевірити наявність змін у репозиторії на Github через web-інтерфейс
  10. Для кожного етапу роботи зробити знімки екрану та додати їх у звіт з описом кожного скіншота
  11. Дати відповіді на контрольні запитання
  12. Зберегти звіт у форматі PDF
  13. Завантажити збережений PDF у репозиторій для лабораторних робіт
