Проектний практикум. Лабораторна робота № 1

Реєстрація на Github та створення репозиторію

Мета роботи

Отримати навички створення профілю на Github та створення власних репозиторіїв

Хід роботи

  1. Створити обліковий запис на Github.com
  2. Створити власний репозиторій, додавши у нього файл README.md
  3. Для кожного етапу роботи зробити знімки екрану та додати їх у звіт з описом кожного скіншота
  4. Дати відповіді на контрольні запитання
  5. Зберегти звіт у форматі PDF
