Проектний практикум. Лабораторна робота № 5

Внесення змін до репозиторію на Github

Мета роботи

Отримати навички внесення змін до локальної робочої копії та репозиторію на Github

Хід роботи

  1. Перевірити глобальні налаштування Git for Windows
  2. Клонувати створений раніше репозиторій як локальну робочу копію
  3. Відредагувати файли локальної робочої копії репозиторію
  4. Перевірити статус локальної робочої копії
  5. Оновити індекс локальної робочої копії, якщо це необхідно
  6. Перевірити статус локальної робочої копії та у випадку відсутності непроіндексованих файлів виконати операцію фіксації
  7. Вивантажити оновлені файли на Github
  8. Перевірити зміст оновлених файлів на Github через web-інтерфейс
  9. Для кожного етапу роботи зробити знімки екрану та додати їх у звіт з описом кожного скіншота
  10. Дати відповіді на контрольні запитання
  11. Зберегти звіт у форматі PDF
